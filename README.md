 全网复制粘贴解放脚本

一键解锁所有网站的复制粘贴功能，解除右键禁用、文字选择禁用、复制禁用。

功能特点
- ✅ 解除右键菜单禁用
- ✅ 解除文字选择禁用
- ✅ 解除复制粘贴剪切禁用
- ✅ 自动移除禁止复制的CSS样式
- ✅ 支持所有主流网站
- ✅ 极简高效，不影响网站性能
- ✅ 兼容Chrome、Edge、Firefox
