# 全网复制粘贴解放脚本

一键解锁所有网站的复制粘贴功能，解除右键禁用、文字选择禁用、复制禁用。

## ✨ 功能特点
- ✅ 解除右键菜单禁用
- ✅ 解除文字选择禁用
- ✅ 解除复制粘贴剪切禁用
- ✅ 自动移除禁止复制的CSS样式
- ✅ 支持所有主流网站
- ✅ 极简高效，不影响网站性能
- ✅ 兼容Chrome、Edge、Firefox

## 🚀 安装使用
1. 安装油猴插件 [Tampermonkey](https://www.tampermonkey.net/)
2. 点击安装脚本：[安装链接](https://greasyfork.org/zh-CN/scripts/你的脚本ID)
3. 刷新网页，即可正常复制

## 📝 支持网站
百度文库、知乎、CSDN、简书、豆瓣、360doc、豆丁网、道客巴巴等几乎所有网站。
